# Bakery Management System 🍰

A desktop-based management application built in **Java** utilizing **Object-Oriented Programming (OOP)** concepts and a user-friendly **Graphical User Interface (GUI)**. This system is designed to streamline bakery operations by managing inventory, tracking daily sales, and processing customer orders efficiently.

## 🚀 Features
* **Interactive GUI:** Built using Java Swing/AWT components for seamless user interaction.
* **Order Management:** Easily take, calculate, and track customer orders in real-time.
* **Inventory Control:** Monitor stock levels for baking ingredients and items.
* **OOP Architecture:** Built with clean, reusable code using inheritance, encapsulation, and polymorphism.
* **Data Persistence:** Uses File I/O to store and retrieve transaction histories and records.

## 🛠️ Technologies Used
* **Language:** Java
* **Libraries:** Java Swing, AWT (Abstract Window Toolkit)
* **Storage:** Java File I/O (Text files/Data serialization)

## 📁 How to Run
1. Clone this repository or download the source files.
2. Open the project in your preferred IDE (NetBeans, Eclipse, or IntelliJ).
3. Run the main class file (usually `Main.java`) to launch the GUI application.
